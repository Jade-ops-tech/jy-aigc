Daily Musings 是一个登录后使用的私密微型日记应用。用户可以记录、编辑、软删除日记，选择心情标签，并安全管理多个 GitHub Token 配置。

## 当前能力

- 邮箱和密码登录，受保护路由统一校验 session。
- 日记创建、倒序信息流、按天分组、编辑和软删除。
- 新建日记草稿在浏览器本地节流保存。
- GitHub Token 使用 AES-256-GCM 加密后存储，支持密钥版本轮换。
- API 使用 tRPC，数据库使用 Drizzle ORM 和 Neon Postgres。
